# ImgDivider

> Divide cualquier imagen en partes perfectas. **100% offline.** Sin servidor, sin trackers, sin límites.

[🔗 Abrir App](https://aaangelmartin.github.io/imgdivider/)

---

## Qué es

**ImgDivider** es una Progressive Web App (PWA) que corre completamente en tu navegador. Sube una imagen, elige un preset o define dimensiones personalizadas, y la app la divide en fragmentos perfectos listos para descargar.

Ideal para:
- Crear **carruseles de Instagram** a partir de una imagen panorámica.
- Generar **grids** de cualquier tamaño.
- Preparar imágenes para **redes sociales** con tamaños exactos.
- Dividir imágenes para **impresión** o cualquier otro uso.

## Características

- **Totalmente offline** — Funciona sin internet tras la primera carga.
- **Sin servidor** — Tu imagen nunca sale de tu dispositivo. Todo el procesamiento ocurre localmente via Canvas API.
- **Presets inteligentes** — Instagram Post, Instagram Story, Instagram Carousel, Pinterest, Twitter/X, YouTube, Facebook, LinkedIn, A4 Print, y grids predefinidos.
- **Modo carrusel** — Detecta automáticamente cuántas partes de 1080×1350 caben en una imagen ancha.
- **Descarga por partes o en ZIP** — Descarga individualmente o todo en un archivo ZIP.
- **Formatos flexibles** — JPEG, PNG, WEBP con control de calidad.
- **Modos de escala** — Recortar, Ajustar, o Estirar.
- **Responsive** — Funciona en desktop, tablet y móvil.
- **Modo oscuro y claro** — Con detección automática del sistema.

## Cómo usar

1. Abre la app en tu navegador (o instálala como PWA).
2. Arrastra o selecciona una imagen.
3. Elige un preset rápido o configura dimensiones personalizadas.
4. Revisa la vista previa con guías.
5. Descarga las partes individuales o en un ZIP.

## Instalación como PWA

En Chrome, Edge, Safari o Firefox:
1. Abre la app.
2. Busca el icono de "Instalar" en la barra de direcciones o menú.
3. Instálala como una app nativa.

Una vez instalada, funciona sin conexión permanente.

## Tecnologías

- **HTML5 + CSS3 + JavaScript (ES6+)** — Vanilla, sin frameworks.
- **Canvas 2D API** — Procesamiento de imágenes 100% cliente.
- **Service Worker + Cache API** — Soporte offline total.
- **JSZip** — Generación de archivos ZIP para descargas.
- **GitHub Pages** — Hosting gratuito y público.
